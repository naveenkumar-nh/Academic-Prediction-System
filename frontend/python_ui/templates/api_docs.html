API Documentation v1

RESTful API for the Academic Performance Prediction System

Authentication Required — All API endpoints require an active session. Login through the web interface first.
Base URL
http://127.0.0.1:5000/api/v1
Students
GET /api/v1/students
List all students with optional search, filter, and pagination.

Query Parameters

ParameterTypeDescription
searchstringSearch by name, reg_no, or department
deptstringFilter by department (CSE, ECE, etc.)
riskstringFilter by risk level (Low, Medium, High)
statusstringFilter by status (Pass, Fail, Pending)
pageintegerPage number (default: 1)
per_pageintegerResults per page (default: 10, max: 100)

Response Example

{ "success": true, "data": [ { "id": 1, "reg_no": "2024CSE001", "name": "John Doe", "dept": "CSE", "year": 2, "academic": { "internal_1": 75.0, "internal_2": 80.0, "internal_3": 70.0, "internal_avg": 75.0, "assignment": 25.0, "attendance": 85.0, "prediction_result": "Pass", "risk_level": "Low" } } ], "pagination": { "page": 1, "per_page": 10, "total": 25, "pages": 3, "has_next": true, "has_prev": false } }

Prediction
POST /api/v1/predict/{reg_no}
Run prediction for a student. Requires academic marks to be entered first. Saves result to prediction history.

Path Parameters

ParameterTypeDescription
reg_nostringStudent register number

Response Example

{ "success": true, "data": { "student_name": "John Doe", "reg_no": "2024CSE001", "prediction_result": "Pass", "risk_level": "Low", "grade": "B", "internal_avg": 75.0, "attendance": 85.0, "suggestions": [ "Try to maintain attendance above 85% for better performance." ] } }
Error Responses
Status CodeDescriptionExample
400 Bad Request {"success": false, "error": "No academic records found."}
401 Unauthorized Redirected to login page
404 Not Found {"success": false, "error": "Student not found"}
